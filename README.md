# DOCKER

## Build the container

> docker build -t getitdone/lillskrot .

## Run the container

> docker run -p 3000:3000 getitdone/lillskrot

## Sign in to docker in the terminal

> docker login
> getitdone
> xxx

## Push the conatiner

> docker push getitdone/lillskrot
